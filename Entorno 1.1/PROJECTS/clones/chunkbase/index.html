Village Info

Village Info is a mod which shows some information about the village you're currently in. It also tells you if there's a village nearby

Last Update: Dec 29,2017

Slime Mod

This mod expands your F3 screen to indicate whether you're in a slime-spawning area. It takes both slime chunks and swamps into account, and also checks the height. Additionally, it displays the numeric world seed.

Last Update: Dec 29,2017

Unglitch

Unglitch is a bugfix mod to prevent entities from glitching through blocks and to reduce client/server desynchronization issues.

Last Update: Sep 23,2013
